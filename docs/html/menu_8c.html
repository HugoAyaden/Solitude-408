int check_menu_click ( SDL_Point mousePos)

Checks if a mouse click occurred on any menu button.

Iterates through all buttons to see if the click position intersects with any button's rectangle. Returns the index of the clicked button or -1 if no button was clicked.

Parameters
mousePosThe position of the mouse click.
Returns
Index of the clicked button (0-4) or -1 if none.
