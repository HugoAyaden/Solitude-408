Displays the game credits with vertical scrolling. 

This function displays a credits screen with a text scrolling effect from bottom to top, similar to movie credits. The background image is displayed behind the text, and certain lines can be highlighted with different font sizes.

The display stops when all credits have scrolled off the screen.
