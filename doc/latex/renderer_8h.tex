Affiche les crédits du jeu avec un défilement vertical. 

Cette fonction affiche un écran de crédits avec un effet de défilement du texte du bas vers le haut, similaire à celui des films (style Star Wars). Un fond d\textquotesingle{}écran est affiché en arrière-\/plan et certaines lignes peuvent être mises en valeur avec une taille de police différente.

L\textquotesingle{}affichage s\textquotesingle{}arrête lorsque tous les crédits sont sortis de l\textquotesingle{}écran.

\hline
\end{DoxyParams}
