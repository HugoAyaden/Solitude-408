void buttons_handleEvent ( SDL_Event * event,
SDL_Window * window )

Gère les événements liés aux boutons.

Cette fonction détecte les clics de souris et active ou désactive les boutons correspondants (portes et lumières).

Paramètres
eventÉvénement SDL reçu.
windowFenêtre SDL utilisée pour récupérer la taille.

◆ drawButton()

void drawButton ( SDL_Renderer * renderer,
TTF_Font * font,
SDL_Rect rect,
int active,
const char * label )

Dessine un bouton interactif avec un texte.

Cette fonction dessine un rectangle représentant un bouton, applique une couleur différente selon son état actif ou non, puis affiche un texte centré à l'intérieur.

Paramètres
rendererLe renderer SDL utilisé pour dessiner.
fontLa police utilisée pour afficher le texte.
rectRectangle définissant la position et la taille du bouton.
activeIndique si le bouton est actif (1) ou non (0).
labelTexte affiché sur le bouton.
