◆ affichage()

void affichage ( camera_type *  camera_type)

Selects appropriate background texture based on game state.

Chooses from dozens of preloaded textures based on combinations of:

  • Battery level (blackout when depleted or light off)
  • Door states (left/right closed)
  • Light states (left/right/general on)
  • Monster proximity to player room Sets camera_type to GAME or delegates to change_camera() when monitor active.
Parameters
camera_typePointer to current camera display mode.

◆ render_credits()

int render_credits ( SDL_Renderer *  renderer,
SDL_Window *  window 
)

Displays the game credits with vertical scrolling.

This function displays a credits screen with a text scrolling effect from bottom to top, similar to movie credits. The background image is displayed behind the text, and certain lines can be highlighted with different font sizes.

The display stops when all credits have scrolled off the screen.

Parameters
rendererThe SDL renderer used for graphical rendering.
windowThe SDL window used to retrieve display dimensions.
